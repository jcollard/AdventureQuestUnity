Detailed Description

An AbstractTextAdventure provides default implementations of almost all of the methods necessary to create a Text Adventure game. To extend this class, you only need to implement the OnStart, GetName, GetAuthor, and GetDescription methods. An example adventure has been provided: DragonsLairAdventure
