string IRoom.GetDescription ( ITextAdventure  adventure)

Given a TextAdventure, returns a description of this room. This method accepts a TextAdventure as a parameter which can be used to change the description depending on the state of the game. This is drawn to the screen when the player first enters the room. This must not be null.
